## Red Flags
<!-- ui-hint: ALERT — any positive checkbox triggers mandatory acknowledgement popup -->
<!-- ui-hint: show reference: Oliveira et al 2018 -->

### Malignancy
- [ ] Unexplained weight loss
- [ ] History of cancer
- [ ] Age >50 with new spinal pain
- [ ] Failure to improve with conservative treatment

### Fracture
- [ ] Significant trauma
- [ ] Prolonged corticosteroid use
- [ ] Osteoporosis confirmed

### Infection
- [ ] Fever
- [ ] Immunosuppressive disease or medication
- [ ] Recent spinal procedure

### Cauda Equina Compression
<!-- ui-hint: URGENT alert if positive — document action taken -->
- [ ] Saddle anaesthesia / perineal symptoms
- [ ] Disturbance of bladder function
- [ ] Disturbance of bowel function
- Action taken if positive::

### Spinal Cord Compression
<!-- ui-hint: URGENT alert if positive -->
- [ ] Bilateral paraesthesia / weakness / sensation changes
- [ ] Gait or balance disturbance
- Action taken if positive::

### Upper Motor Neurone Signs
<!-- ui-hint: populate from physical examination — cross-referenced field -->
- [ ] Hyperreflexia (ankle jerk bilaterally)
- [ ] Positive Babinski reflex (upgoing plantars)
- [ ] Clonus (calf bilaterally)
- [ ] Impaired balance on Romberg's test
- [ ] Impaired coordination (finger-nose, alternating forearm, heel-shin)
- Interpretation::

## Differential Screening

### Ankylosing Spondylitis
- [ ] Insidious onset
- [ ] Lumbar / SIJ progressing to other areas
- [ ] Inflammatory symptom pattern
- [ ] Breathing difficulties
- [ ] Fever / weight loss
- Likelihood:: [ ] Low [ ] Moderate [ ] High — action::

### Abdominal Aortic Aneurysm
- [ ] Pulsating lumbar / groin pain
- [ ] Age >50
- [ ] Cardiovascular risk factors present
- [ ] Sudden onset with low BP / shock (ruptured)
- Likelihood:: [ ] Low [ ] Moderate [ ] High — action::

### Vascular Claudication
- [ ] Non-dermatomal leg symptoms
- [ ] Age >50
- [ ] Cardiovascular risk factors
- [ ] Pain / burning / fatigue with walking, relieved by rest
- [ ] Peripheral vascular signs (cold, blotchy, shiny, hairless skin)
- [ ] Impotence in men
- [ ] Leg pain at night
- Likelihood:: [ ] Low [ ] Moderate [ ] High — action::
