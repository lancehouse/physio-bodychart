━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Run this Python script in 4 positions to determine
axis mapping for this specific device. Each run takes
~1 second. Note results for each position.

python3 -c "
import time, math
dev = '/sys/bus/iio/devices/iio:device0'

def read(axis):
    with open(f'{dev}/in_accel_{axis}_raw') as f:
        return float(f.read().strip())

samples = 20
print('Taking 20 readings, hold still...')
xs, ys, zs = [], [], []
for i in range(samples):
    xs.append(read('x'))
    ys.append(read('y'))
    zs.append(read('z'))
    time.sleep(0.05)

x = sum(xs)/samples
y = sum(ys)/samples
z = sum(zs)/samples
print(f'X avg: {x:.1f}')
print(f'Y avg: {y:.1f}')
print(f'Z avg: {z:.1f}')
print(f'Angle (sagittal): {math.atan2(math.sqrt(x*x+z*z),y)*180/math.pi:.1f} deg')
print(f'Angle (frontal):  {math.atan2(math.sqrt(y*y+z*z),x)*180/math.pi:.1f} deg')
"

Record X/Y/Z averages for each position.
The axis that changes most between positions 1 and 3

NOISE FILTERING — REQUIRED BEFORE DISPLAY
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Raw values jump ±50 counts at rest. This is normal
HID sensor behaviour. Rolling average of 20 samples
at 20Hz gives clinically acceptable stability (±0.5°)
with ~1 second settling time. Do not display raw values.

C implementation — rolling average ring buffer:

#define ACCEL_SAMPLES 20
#define ACCEL_DEVICE "/sys/bus/iio/devices/iio:device0"

static double x_buf[ACCEL_SAMPLES];
static double y_buf[ACCEL_SAMPLES];
static double z_buf[ACCEL_SAMPLES];
static int buf_idx = 0;
static int buf_filled = 0;
static double accel_scale = 1.0;  /* read once at startup */

static double read_accel_axis_raw(const char *axis) {
    char path[256];
    snprintf(path, sizeof(path),
        ACCEL_DEVICE "/in_accel_%s_raw", axis);
    FILE *f = fopen(path, "r");
    if (!f) return 0.0;
    double val;
    fscanf(f, "%lf", &val);
    fclose(f);
    return val * accel_scale;
}

static void accel_read_scale(void) {
    FILE *f = fopen(ACCEL_DEVICE "/in_accel_scale", "r");
    if (!f) return;
    fscanf(f, "%lf", &accel_scale);
    fclose(f);
}

static double smoothed_angle_sagittal(void) {
    x_buf[buf_idx] = read_accel_axis_raw("x");
    y_buf[buf_idx] = read_accel_axis_raw("y");
    z_buf[buf_idx] = read_accel_axis_raw("z");
    buf_idx = (buf_idx + 1) % ACCEL_SAMPLES;
    if (buf_filled < ACCEL_SAMPLES) buf_filled++;

    double ax = 0, ay = 0, az = 0;
    for (int i = 0; i < buf_filled; i++) {
        ax += x_buf[i];
        ay += y_buf[i];
        az += z_buf[i];
    }
    ax /= buf_filled;
    ay /= buf_filled;
    az /= buf_filled;

    /* NOTE: axis mapping below is a placeholder.
       Run the axis orientation test and update
       the formula with correct axis assignment
       for this specific device before building. */
    return atan2(sqrt(ax*ax + az*az), ay)
           * 180.0 / G_PI;
}

static double smoothed_angle_frontal(void) {
    /* Same buffer — call sagittal first each tick
       to populate buffer, then call this for the
       alternate plane calculation */
    double ax = 0, ay = 0, az = 0;
    for (int i = 0; i < buf_filled; i++) {
        ax += x_buf[i];
        ay += y_buf[i];
        az += z_buf[i];
    }
    ax /= buf_filled;
    ay /= buf_filled;
    az /= buf_filled;

    /* NOTE: update axis mapping after orientation test */
    return atan2(sqrt(ay*ay + az*az), ax)
           * 180.0 / G_PI;
}

Polling timer — call at 20Hz (every 50ms):

static gboolean accel_poll_cb(gpointer user_data) {
    InclinometerState *state = user_data;
    double angle;

    if (state->plane == PLANE_SAGITTAL)
        angle = smoothed_angle_sagittal();
    else
        angle = smoothed_angle_frontal();

    /* Angle relative to zero reference */
    double relative = angle - state->zero_reference;

    /* Update stability indicator */
    state->stability = compute_stability();

    /* Update display */
    char label[32];
    snprintf(label, sizeof(label), "%.1f°",
             fabs(relative));
    gtk_label_set_text(state->angle_label, label);
    update_stability_bar(state);

    return G_SOURCE_CONTINUE;
}

Stability calculation — standard deviation of
last 10 readings. Capture is reliable when StdDev < 1°:

static double compute_stability(void) {
    int n = MIN(buf_filled, 10);
    if (n < 2) return 99.0;
    double sum = 0, sum_sq = 0;
    for (int i = 0; i < n; i++) {
        /* use whichever axis is primary for current plane */
        double v = y_buf[(buf_idx - 1 - i +
                          ACCEL_SAMPLES) % ACCEL_SAMPLES];
        sum += v;
        sum_sq += v * v;
    }
    double mean = sum / n;
    double variance = (sum_sq / n) - (mean * mean);
    return sqrt(fabs(variance));
}
